## 🔧 Troubleshooting

### Issue: Tests fail with import errors
**Solution:**
```powershell
cd june
pip install langdetect
```

### Issue: Hindi not recognized
**Solution:**
- Ensure Google Cloud credentials are valid
- Check `alternative_language_codes` includes `"hi-IN"`
- Speak clearly with natural pauses

### Issue: Wrong voice used
**Solution:**
- Check `auto_language_switching: true` in config
- Verify `voice_preferences` section exists
- Review logs with `--verbose` flag

## 🎯 Best Practices

### **For Maximum Accuracy:**
1. ✅ Use enhanced config: `config-enhanced-multilingual.json`
2. ✅ Speak naturally (not too fast or slow)
3. ✅ Use good microphone in quiet environment
4. ✅ Start with pure Hindi/English, then try Hinglish
5. ✅ Add your common phrases to `speech_contexts`

### **For Natural Hinglish:**
1. ✅ Mix naturally: "Mujhe ek coffee chahiye"
2. ✅ Keep tech terms in English: "app", "laptop"
3. ✅ Use common Hindi words: "main", "aap", "kaise"
4. ❌ Avoid: Rapid language switching mid-sentence
