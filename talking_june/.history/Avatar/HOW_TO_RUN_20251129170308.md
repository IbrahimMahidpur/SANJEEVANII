## 🐛 Problem Solving

### Problem 1: "Port already in use"
**Solution:**
```cmd
netstat -ano | findstr "8765"
netstat -ano | findstr "8001"
netstat -ano | findstr "8080"
```
Agar koi process chal rahi hai, usko band karo ya ports change karo.

### Problem 2: "Connected to June VA bridge" nahi dikh raha
**Solution:**
1. Bridge Server check karo (Terminal 1)
2. Browser refresh karo (Ctrl+R)
3. Console clear karke phir se dekho

### Problem 3: June "Listening for sound..." se aage nahi ja raha
**Solution:**
1. Microphone permissions check karo
2. **Zor se bolo** (volume thoda high chahiye)
3. Quiet jagah mein test karo (noise kam ho)

### Problem 4: Lip-sync nahi ho rahi
**Solutions:**
1. Browser console dekho - koi error hai?
2. Bridge server dekho - WebSocket message ja raha hai?
3. June terminal dekho - `[Bridge] ✅ Sent audio URL...` dikh raha hai?
