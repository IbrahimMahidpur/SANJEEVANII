# STT Hindi Detection Fix

## Problem
STT was not detecting Hindi/Hinglish properly:
- Primary language was `en-US` (English)
- Hindi was only in alternatives, so English was always tried first
- Even when Hindi words were present, system defaulted to English

## Root Cause
```json
"primary_language": "en-US",  // ❌ WRONG - English first!
"alternative_language_codes": ["hi-IN", "en-IN"],  // Hindi as backup only
```

## Solution

### 1. Changed Primary Language to Hindi
```json
"primary_language": "hi-IN",  // ✅ CORRECT - Hindi first!
"alternative_language_codes": ["en-IN", "bn-IN", "en-US"],  // English as backup
```

### 2. Added Speech Contexts for Hindi Words
```json
"speech_contexts": {
    "phrases": [
        "kya", "hai", "ho", "main", "mujhe", "kar", "karo",
        "chahiye", "batao", "nahi", "haan", "theek", "achha",
        "help", "kaise", "joke", "suno", "bolo", "samajh"
    ],
    "boost": 20.0  // 20x boost for these phrases
}
```

### 3. Improved STT Logic with Hindi Word Detection
**Before:**
- Only checked for Devanagari script
- If no script, assumed English

**After:**
- Tries Hindi FIRST (primary_language = hi-IN)
- Checks for Devanagari script OR transliterated Hindi words
- If 2+ Hindi words found → confirms as Hindi/Hinglish
- Only tries alternatives if no Hindi indicators

**New Detection Flow:**
```
1. Try hi-IN (primary) → "Mujhe help chahiye"
2. Check: Devanagari? NO
3. Check: Hindi words? YES (Mujhe=1, chahiye=2) → 2 words
4. ✅ CONFIRMED Hindi/Hinglish
5. Return: (transcript, "HI-IN")
```

**Old Flow (broken):**
```
1. Try en-US (primary) → "2 months"
2. ✅ Got result → return immediately
3. Never tried Hindi alternatives
4. ❌ Missed Hindi detection
```

## Key Changes

### Config (`june/config.json`)
```diff
- "primary_language": "en-US",
- "alternative_language_codes": ["hi-IN", "en-IN"],
+ "primary_language": "hi-IN",
+ "alternative_language_codes": ["en-IN", "bn-IN", "en-US"],
+ "speech_contexts": {
+     "phrases": [...Hindi words...],
+     "boost": 20.0
+ }
```

### STT Logic (`june_va/models/stt.py`)
```python
# NEW: Uses LanguageDetector to validate
from june_va.language_detector import LanguageDetector

# NEW: Counts Hindi words in transcript
hindi_word_count = LanguageDetector.count_hindi_words(transcript)

# NEW: Confirms Hindi if 2+ Hindi words
if hindi_word_count >= 2:
    logger.info(f"✓ CONFIRMED Hindi/Hinglish ({hindi_word_count} Hindi words)")
    return (transcript, "HI-IN")
```
